
# 🤖 Rule-Based AI Chatbot

## DecodeLabs Industrial Training Kit – AI Project 1

A simple and interactive Rule-Based AI Chatbot developed using HTML, CSS, and JavaScript. The chatbot responds to predefined user inputs using if-else decision-making logic and demonstrates fundamental Artificial Intelligence concepts.

### 👨‍💻 Developed By
**Abdul Aakhir Saifullah**

---

## 📌 Project Overview

This chatbot was created as part of the DecodeLabs Industrial Training Kit (AI Project 1). The objective of the project is to build a chatbot that can interact with users through predefined rules rather than machine learning models.

The chatbot continuously accepts user input and generates responses based on programmed conditions.

---

## 🚀 Features

- Greeting Recognition (hello, hi, hey)
- Exit Commands (bye, exit)
- Help Command
- Rule-Based Responses
- Continuous User Interaction
- Interactive Web Interface
- Sidebar with Suggested Commands
- Responsive and User-Friendly Design

---

## 🛠 Technologies Used
